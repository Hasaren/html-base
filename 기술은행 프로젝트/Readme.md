# 기술 정보 검색 웹 페이지

## 1. 프로젝트 소개

본 프로젝트는 사용자가 브라우저에서 기술 관련 정보를 검색하고, 카테고리별로 탐색할 수 있도록 구성한 웹 페이지입니다.

메인 화면에는 헤더 이미지, 검색창, 상단 메뉴, 기술 통계 카드, 검색 카테고리 버튼 등을 배치하였습니다.

---

## 2. 사용 데이터 출처

| 구분 | API명 / 데이터명 | URL | 설명 |
|---|---|---|---|
| 이미지 | 로컬 이미지 파일 | `images/header-bg.jpg` | 메인 헤더 배경 이미지 |
| 아이콘 | Font Awesome | https://fontawesome.com | 언어 버튼 및 검색 아이콘 표시 |
| API | 한국산업기술진흥원_기술은행 수요기술 조회 서비스_GW | https://www.data.go.kr/data/15158929/openapi.do#tab_layer_detail_function | 기술 통계 및 카테고리 정보 표시 |

※ 현재 코드 기준으로 외부 API 호출 기능은 확인되지 않았습니다.  
추후 API를 연동할 경우 사용한 API명과 URL을 추가하면 됩니다.

## 3. 구현 기능 요약

- 전체 화면 헤더 영역 구성
- 배경 이미지와 어두운 오버레이 적용
- 로고 및 상단 내비게이션 버튼 배치
- 언어 선택 버튼 UI 구현
- 상단 아코디언 메뉴 구현
- 메인 검색창 UI 구현
- 기술 통계 카드 그리드 구성
- 검색 카테고리 버튼 3x3 배치
- 푸터 영역 구성
- CSS Grid와 Flexbox를 활용한 레이아웃 구현

## 5. 주요 화면 구성

### 헤더 영역

- 전체 화면 높이 `100vh`
- 배경 이미지 적용
- 검은색 반투명 오버레이 적용
- 중앙에 제목과 검색창 배치

### 내비게이션 영역

- 왼쪽 로고
- 오른쪽 버튼 메뉴
- Flexbox를 사용하여 양쪽 정렬

### 검색 영역

- 투명 배경 검색창
- 입력창과 버튼을 가로 정렬
- 검색 아이콘 버튼 사용 가능

### 기술 통계 영역

- 4개의 카드가 한 줄에 배치되는 Grid 구조
- 통계명, 값, 단위 표시

### 카테고리 영역

- 3행 3열 버튼 구조
- 카테고리별 검색 진입용 UI

## 6. 브라우저 실행 방법

별도의 서버 실행 없이 HTML 파일을 브라우저에서 직접 열어 실행할 수 있습니다.

### 실행 방법

1. 프로젝트 폴더를 엽니다.
2. `index.html` 파일을 찾습니다.
3. 파일을 더블 클릭하여 브라우저에서 실행합니다.

또는 VS Code 사용 시:

1. VS Code에서 프로젝트 폴더 열기
2. `index.html` 파일 열기
3. Live Server 확장 프로그램 실행
4. 브라우저에서 자동 실행 확인
